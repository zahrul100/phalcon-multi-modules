# Menggunakan Phalcon-multi-modules
## Struktur direktori PHALCON SKELETON

```

`apps
    config
          config.php
          loader.php
          modules.php
          routing.php
          services.php
    lib
         Phalcon
    modules
          dashboard
            Domain Driven Design
    .env
    bootstrap.php
public
    assets
    .htaccess
    index.php
.htaccess
composer.json`

```

## Persiapan
### composer install
```composer install```
### Membuat folder cache
Buat folder cache pada direktori apps
``` mkdir apps/cache ```
lalu ubah permision cache,```chmod 777 apps/cache```
